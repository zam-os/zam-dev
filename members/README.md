# Community Members

This directory documents the membership of this community — who belongs, what
their role is, and how new members can join.

## Member roles

| Role | Description |
|------|-------------|
| `maintainer` | Shapes direction, approves changes to beliefs and goals |
| `contributor` | Actively builds — submits code, docs, or other work |
| `member` | Participates in learning and community activities |
| `developer` | Developer communities: clones source repos, uses npm link |

## Member files

Each member has a file named `<user_id>.md`:

```markdown
---
user_id: thomas
role: developer
joined: 2026-03-28
personal_repo: github:Josefczak/zam-Thomas
---

Optional notes about this member's focus or responsibilities.
```
